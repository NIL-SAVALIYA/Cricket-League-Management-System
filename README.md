# 🏏 Cricket League Management System

An Advanced Cricket League Management System built with **Node.js**, **Express.js**, **Prisma ORM**, **PostgreSQL**, and **React.js**.

---

## 🚀 Features

- User Authentication (JWT)
- Role Based Access Control
- Team Management
- Player Management
- Tournament Management
- Match Scheduling
- Live Scoring
- Points Table
- Statistics Dashboard
- Venue Management
- Organizer Dashboard
- Admin Panel

---

## 🛠 Tech Stack

### Backend

- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- JWT
- Zod
- bcrypt

### Frontend

- React.js
- Vite
- Tailwind CSS
